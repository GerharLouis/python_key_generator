#  Python Offline Key Generator

An open-source **offline license key generator** and manager with GUI (Tkinter + SQLite).  
Ideal for software developers who want to protect and manage offline license keys.

---

## Features
- Offline license creation & verification  
- Hardware-locked license generation  
- Local SQLite database  
- Export `.key` files  
- Email license keys directly  
- Batch license generation via CLI
